# ngrok

npm install -g ngrok

npm run dev 를 실행하고(로컬 서버를 실행한다.)

실행 명령어 'ngrok http 8080' (포트번호)

Forwarding                    http://ae98f6e6.ngrok.io -> localhost:8080                                                     
Forwarding                    https://ae98f6e6.ngrok.io -> localhost:8080 

<br>

위의 주소(http~, https~)로 브라우저에 접속하면 된다.
