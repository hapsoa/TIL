# regular expression(정규표현식)

## 정규표현식 만들기

```javascript
var re = /ab+c/;
var re = new RegExp('ab+c');
```

| 정규표현식 | 설명                                             |
| ---------- | ------------------------------------------------ |
| ^x         | 문자열이 x로 시작합니다.                         |
| x\$        | 문자열이 x로 끝납니다.                           |
| .x         | x가 마지막으로 끝나고 앞에 임의의 한 문자가 있음 |
| x+         | x가 1번이상 반복합니다.                          |
| x?         | x가 존재하거나 존재하지 않습니다                 |
| x\*        | x가 0번이상 반복합니다                           |
