# 스타일 가이드

## 우선순위 A 규칙: 필수

### # 컴포넌트 이름에 합성어 사용

root 컴포넌트인 App 컴포넌트를 제외하고 컴포넌트의 이름은 항상 합성어를 사용하여야 합니다.
<br>

html엘리먼트와 충돌 방지를 위해

```javascript
Vue.component('todo-item', {
    // ...
})

export default {
    name: 'TodoItem',
    // ...
}
```

<br>

### # 컴포넌트 데이터

컴포넌트의 data 는 반드시 함수다.
<br>

컴포넌트(i.e. new Vue 를 제외한 모든곳)의 data 속성은 값으로 오브젝트를 반환하는 함수여야 합니다.

```javascript
data: function () {
    return {
        listTitle: '',
        todos: []
    }
}
```
